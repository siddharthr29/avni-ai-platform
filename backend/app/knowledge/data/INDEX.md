## Key Knowledge for AI Platform

### SRS Tab Structure (from videos)

The official Avni scoping document follows this tab order:
1. **Help & Status Tracker** — Guide to filling other sheets
2. **Program Summary** — Location, data systems, rollout date, user count
3. **Program Detail** — Per program: name, objective, eligibility, entry/exit, beneficiaries, success indicators, forms list, reports
4. **User Persona** — User types, descriptions, counts
5. **W3H** — What (activities), When (timing), Who (which user), How (mobile/web), Forms to be Scheduled
6. **Forms** — Per form: Page Name, Field Name, Data Type, Mandatory, User/System, Numeric validation, Units, Date type, Single/Multi select, Unique options, Skip logic
7. **Visit Scheduling** — On completion of, Schedule form, Frequency, Schedule for, Conditions, Schedule date, Overdue, Cancellation, Weekend handling, On edit
8. **Offline Dashboard Cards** — Card name, Logic, User type
9. **Permissions** — Per form × per user group: View, Register, Edit, Void
